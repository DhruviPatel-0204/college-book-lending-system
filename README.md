# College Book Lending System

## Core Features
### IITR-Restricted Access
- Ensures platform use is exclusive to the campus community.
### Book Discovery
- Easily browse and search for available non-academic books.
### Personal Library Management
- Add and list your own books for others to borrow.
### Seamless Exchange
- Streamlined borrow request and approval system.
### "My Books" Page
- Track all your listings and borrowing requests in one place.

## Tech Stack
### Frontend
- React + Tailwind CSS for a modern, responsive user interface.
### Backend
- Node.js for robust API development and server logic.
### Database
- Supabase for flexible and scalable data storage.
### Hosting
- Vercel for reliable deployment and availability.
### Authentication
- IITR email restriction with planned OTP verification for security.

Website link: https://college-book-lending-system.vercel.app/
